
　　『 深層 [ sublim.com ] 』

　◎　機　能

　　幸せを感じるようになる．


　◎　使い方

　　A>sublim [-r]
　　
　　“sublim”で常駐，“sublim -r” で常駐解除です．

　◎　注　意

　　　このソフトは垂直同期割り込みだけを乗っ取っています．
　　　そのため，Ｍｉｆｅｓなどのソフトを実行するなどして， CRTCからの
　　割り込み信号が発生しなくなった場合，sublim.comは機能しなくなります．
　　　そのときは，もう一度“sublim”を実行してください．　CRTCがリセッ
　　トされ，再び sublim.com が機能しだします（２重常駐はしません）．

　　るのですが，なんという法律なのでしょうか？


　◎　転載について．

　　このソフトおよびドキュメントの改変，転載，その他もろもろは
　　なんでもアリアリのアリです．
　　みんなで幸せになりましょう．
